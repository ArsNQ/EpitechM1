Welcome to the Count of Money project, 

you need to follow step by step all the following steps to run the project correctly.

# 1 - Install Docker

You need docker to run the project, install it first :
Windows :
https://docs.docker.com/docker-for-windows/install/

Ubuntu: 
https://doc.ubuntu-fr.org/docker

MacOS:
https://docs.docker.com/docker-for-mac/install/

# 2 Execute Docker
All the project run under containers on Docker, so you just need to run the following commands and the project will work !

docker-compose build
docker-compose up -d

Then you need to go on your favorite browser, and need to go on the following address :
http://127.0.0.1:8080/

Congratulation, you run the count of money project! 

Enjoy it
